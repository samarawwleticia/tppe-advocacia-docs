## Fluxos

### [Landing Page Web / Landing Page Mobile]

Essa é a Landing Page, onde será possível que usuários acessem informações sobre o escritório e o cliente, seus diferenciais, áreas de atuação, artigos e formas de entrar em contato. Possui uma tela principal contendo a barra de navegação onde o usuário pode acessar facilmente diferentes tópicos.

**Telas:** Tela Landing Page 

---

### [Sistema Web/ Sistema Mobile]

Essa é o Sistema que se inicia na aba usuários, que mostra um painel de gerenciamento de usuários. Na barra de navegação, é possível acessar ferramentas para edição da Landing Page e as telas de Artigos, Leads, Clientes, Agenda, Processos, Tarefas e painel de Notificações. Mais detalhes podem ser encontrados dentro do protótipo no link listado acima. 

> Obs: O protótipo do sistema mobile se inicia na área de login e acessa o sistema descrito acima ao clicar no botão 'Entrar'. 

**Telas:** 

    - Tela Usuários → Tela Novo Usuário → Tela Registros
    - Tela Edição Landing Page 
    - Tela Artigos → Tela Novo Artigo
    - Tela Leads → Tela Informações Lead 
    - Tela Clientes → Tela Adicionar Cliente 
    - Tela Agenda
            → Tela Calcular Prazo 
            → Tela Novo Compromisso
    - Tela Processos
            → Tela Novo Processo
            → Tela Abrir Ficha do Processo
    - Tela Tarefas
            → Tela Nova Tarefa
            → Tela Edição de Tarefa
    - Tela Notificações

---

### [Login Web]

Tela que representa o ambiente de login, acessado quando o usuário quiser entrar na página de Sistema. No protótipo, o sistema é acessado ao acionar o botão 'Entrar'.

**Telas:** Tela Login
